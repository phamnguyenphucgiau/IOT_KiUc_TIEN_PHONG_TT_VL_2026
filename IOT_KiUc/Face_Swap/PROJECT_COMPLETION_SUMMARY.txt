🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "Module not found" error
Solution: Run setup_unified_pipeline.bat

Problem: "CUDA out of memory"
Solution: Use batch_size=1, resolution=256, close other apps

Problem: "FFmpeg not found"
Solution: 
  Windows: choco install ffmpeg
  Linux: sudo apt-get install ffmpeg
  Mac: brew install ffmpeg

Problem: Models not found
Solution: Download from GitHub and extract to checkpoints/ folder

Problem: Slow processing
Solution: Enable GPU (install NVIDIA drivers), use batch_size=2

For more help:
  python pipeline_diagnostics.py
  Read: UNIFIED_PIPELINE_README.md
